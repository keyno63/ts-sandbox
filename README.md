# ts-sandbox

## Motivation
TypeScript を試してみる

## Usage

Use `npm` command for running this repository.

to install dependencies.
```shell
npm run install
```

To transpile typescript to javascript.
```shell
npm run build
```

To lint.
```shell
npm run lint
```

To run.
```shell
npm run run
```
